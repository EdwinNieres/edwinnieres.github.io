Mi Portafolio

Pagina Cupon

En este proyecto recreamos mediante una maqueta una pagina web usando como principal herramientas HTML para el maquetado, un poco de CSS para los estilos y bootstrap para realizar basicamente el nav-bar, los productos y el footer

Pagina Iguana

En esa oportunidad hicimos una pagina web con Flexbox usando como muestra una maqueta en XD, donde tambien aprendimos a usar Iconos. La pagina web debia tener 6 secciones con algunas imagenes de iguanas, sus caracteristicas y sus distintos colores y variaciones

Pagina Landing

Este practicamente fue nuestro primer o segundo proyecto en la academia, donde realizamos una landing page personalizada usando etiquetas de tablas para la sección principal y un poco de CSS para la personalización. Por primera vez hicimos una landing page
